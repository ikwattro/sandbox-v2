2. Task: Organizations with offices in Paris


Find all organizations that have sites with offices located in Paris and return their names!

Use the described pattern and filter by location.

Solution on next slide.

2. Solution: Organizations with offices in Paris


Pattern

Cypher

MATCH (o:Org)-[:HAS_SITE]->(:Office)-[:IS_LOCATED_IN]->(c:City)
WHERE c.name = "Paris"
RETURN o.name

3. Solution: Countries with "GraphIT" offices


Pattern

Cypher

MATCH (o:Org {name:"GraphIT"})-[:HAS_SITE]->(office:Office),
      (office)-[:IS_LOCATED_IN]->(:City)-[:IS_IN]->(c:Country)
RETURN c.name

3. Task: Colleagues of Daniel


Find all people that work in the same department as Daniel and return their names!

MATCH ... (pattern) ...
WHERE ...
RETURN ..

Use the described pattern and filter by name.

Solution on next slide.

4. Solution:Colleagues of Daniel


Pattern

Cypher

MATCH (p:Person)-[:WORKS_AT]->(d:Dept),
      (d)<-[:WORKS_AT]-(b:Person)
WHERE p.name = "Daniel"
RETURN b.name

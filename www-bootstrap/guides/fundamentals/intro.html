From Tables to Patterns


What graph pattern are we looking for?

The pattern in Cypher:

(p:Person)-[:WORKS_AT]->(d:Dept)

From SQL to Cypher


MATCH allows you to find a pattern in the graph

MATCH (p:Person)-[:WORKS_AT]->(d:Dept)
RETURN p.name, d.name

WHERE allows you to filter the results

MATCH (p:Person)-[:WORKS_AT]->(d:Dept)
WHERE d.name = "IT-Department"
RETURN p.name, d.name

SQL vs. Cypher


SELECT * FROM Person
LEFT JOIN Person_Dept
  ON Person.Id = Person_Dept.PersonId
LEFT JOIN Dept
  ON Department.Id = Person_Dept.DeptId
WHERE Dept.name = "IT-Department"
MATCH (p:Person)-[:WORKS_AT]->(d:Dept)
WHERE d.name = "IT-Department"
RETURN p.name, d.name
